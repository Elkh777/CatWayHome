# 🐱 CatWayHome

> 2D-игра о приключениях потерявшегося кота, который ищет путь домой.

## 📖 О игре
**CatWayHome** — это 2D-платформер, в которой главный герой — кот, случайно оказавшийся далеко от родного дома. Игроку предстоит провести его через разнообразные локации, избегая препятствий,  и находя безопасный маршрут обратно. С каждым уровнем сложность возрастает: появляются новые типы препятствий.
## 🏗 Архитектура проекта
Проект реализован с использованием паттерна **MVC (Model-View-Controller)**.  

### Как MVC применён в игре:
- **🔹 Model** — чистая логика: состояние кота (позиция, здоровье, инвентарь), данные уровня, физика столкновений, система сохранения прогресса. Не зависит от графики и ввода.
- **🔹 View** — отрисовка: рендеринг спрайтов, анимации, UI-элементы (меню, HUD), эффекты частиц, работа камеры. Только читает данные из Model и отображает их.
- **🔹 Controller** — управление: обработка нажатий клавиш/геймпада, координация Model и View, управление состояниями игры (меню → уровень → пауза → победа/проигрыш).

## 🎮 Управление
| Клавиша | Действие |
|---------|----------|
| `← →` / `A D` | Движение влево / вправо |
| `↑` / `W` / `Пробел` | Прыжок |
| `Esc` / `P` | Пауза / Меню |
